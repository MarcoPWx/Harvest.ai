# Swagger Live vs Snapshot Fallback (Flow)

<Meta title="Flows/Swagger Live vs Snapshot Fallback" />

This flow summarizes how the Swagger UI story tries the live OpenAPI endpoint and falls back to the snapshot automatically.

See also:

- <a href="?path=/docs/docs-swagger-ui--docs">Docs → Swagger UI</a>
- <a href="/doc?path=docs/api/SWAGGER_FALLBACK.md#behavior" target="_blank" rel="noreferrer">

## Steps

- Start dev server: `npm run dev:mock` (serves /api/openapi.json)
- Open Docs/Swagger UI
- If live fails, the story loads /docs/api/openapi.snapshot.json

See the doc in the Help panel for details.
