These playgrounds run in your Next.js dev server and are embedded here via iframes. Make sure your dev server is running on <code>http://localhost:3002</code>.

## How to run

```bash
npm run dev:mock  # Next.js dev at http://localhost:3002
npm run storybook # Storybook at http://localhost:6006
```

> Tip: You can also open the playgrounds directly in the browser: <code>http://localhost:3002/playground</code>.
